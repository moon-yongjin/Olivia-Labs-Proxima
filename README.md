# ⚡ Proxima - 진돗개 에디션 (Tuned Version) 🐕

### ✅ 주요 변경 사항
- **CLI 기능 도입**: 터미널에서 `proxima` 명령어로 즉시 AI 제어 가능
- **Grok 버그 수정**: 입력 오류 및 팝업 방해 로직 완벽 해결
- **한국어 대응**: 국내 사용자 환경에 맞춘 UI 인식 및 응답 캡처 최적화
- **자동 빌드**: GitHub Actions를 통한 윈도우용 `.exe` 자동 배포 설정
